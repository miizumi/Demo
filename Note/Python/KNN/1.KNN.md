# K Nearest Neighjbor 最近鄰居法


KNN最近鄰居法也稱為 K-近鄰算法。KNN主要用於分類，也是所有機器學習法中最簡單，使用最廣的演算法之一。

KNN原理為透過K個最近的鄰居的分類，賦予對象類別。

也就是透過附近K個鄰居做多數表決，K通常會是3、5，而且絕對不會用偶數，使用偶數可能會出現$1/2$的尷尬情況，如果看到有人使用偶數，那我們可以笑他。
